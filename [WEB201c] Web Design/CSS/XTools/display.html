Hướng dẫn CSS Display Properties

1. display: block (Giá trị mặc định cho div, p, h1-h6)

Block Element 1
Block Element 2
Block Element 3

⭐ Đặc điểm:
- Luôn bắt đầu trên một dòng mới
- Chiếm toàn bộ chiều ngang có sẵn
- Có thể set width và height
- Các phần tử block mặc định: div, p, h1-h6, section, article

2. display: inline (Giá trị mặc định cho span, a, img)

Inline 1 Inline 2 Inline 3

⭐ Đặc điểm:
- Không bắt đầu trên dòng mới
- Chỉ chiếm không gian cần thiết
- KHÔNG thể set width và height
- Các phần tử inline mặc định: span, a, img, em, strong

3. display: inline-block

Box 1
Box 2
Box 3

⭐ Đặc điểm:
- Kết hợp tính năng của block và inline
- Không bắt đầu trên dòng mới
- CÓ THỂ set width và height
- Thường dùng cho: buttons, navigation items

4. display: flex (Modern Layout)

Flex 1
Flex 2
Flex 3

⭐ Đặc điểm:
- Tạo layout một chiều (row hoặc column)
- Dễ dàng căn chỉnh items (justify-content, align-items)
- Linh hoạt trong việc chia không gian
- Thường dùng cho: navigation bars, card layouts, centering content

⭐ Đặc điểm:
- Tạo layout hai chiều (rows và columns)
- Dễ dàng tạo grid layouts phức tạp
- Kiểm soát chính xác kích thước và vị trí items
- Thường dùng cho: photo galleries, dashboard layouts, complex grid systems

6. display: none

Element này sẽ ẩn khi click button

⭐ Đặc điểm:
- Ẩn hoàn toàn element khỏi trang
- Không chiếm không gian trên trang
- Khác với visibility: hidden (vẫn giữ không gian nhưng ẩn)
- Thường dùng cho: toggle menus, modals, responsive design
